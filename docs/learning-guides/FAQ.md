# 🔍 Frequently Asked Questions

## General

**Q: What is AlexBot?**
A: AlexBot is Alex's digital twin running on OpenClaw - a multi-agent personal assistant platform.

**Q: What can I ask?**
A: Anything! Technical questions, architecture, how things work. But some internal details stay private (file paths, people profiles, etc.)

**Q: Why do you sometimes not respond?**
A: I follow specific rules about when to speak in groups vs DMs. See [05-security-boundaries.md](05-security-boundaries.md)

## Technical

**Q: What model are you using?**
A: Default is Claude Sonnet 4.5, but different agents use different models. See [08-multi-agent.md](08-multi-agent.md)

**Q: What's the difference between temperature settings?**
A: See [01-model-parameters.md](01-model-parameters.md#temperature)

**Q: How do I write better prompts?**
A: See [02-prompt-engineering.md](02-prompt-engineering.md)

A: Security policy - don't expose internal structure. See [05-security-boundaries.md](05-security-boundaries.md#what-i-cant-share)

## Playing Group

**Q: How does scoring work?**
A: See [09-scoring-system.md](09-scoring-system.md)

**Q: What makes a high-scoring challenge?**

## Learning Group

**Q: Why are your answers sometimes short?**
A: Alex set a 30-sentence max to save tokens. I reference these guides instead of writing long explanations.

**Q: How do I index/check materials?**
A: Use memory_search tool or check specific guide files in this repo.

**Q: Can you explain [technical topic]?**
A: Yes! Check the relevant guide or ask and I'll create/update one.
